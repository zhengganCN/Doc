# 探查image信息

## 说明

![API说明](C:\Users\93281\Desktop\af5ec3b7-9e3e-455e-ae5e-4018463ae06e.png)

在介绍容器操作命令时，我们提到过获取容器详细信息的API，这里我们再介绍一个获取镜像详细信息的API。与容器相比，镜像信息要少很多。

因为镜像是作为模板来使用的，没有runtime数据，所以它的信息量就很小了。
