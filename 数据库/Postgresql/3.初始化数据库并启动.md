# 初始化数据库并启动

安装postgresql后不能直接启动，必须先用postgres用户初始化数据库后才能启动postgresql。

## 初始化数据库

``` shell
# 切换至postgres用户
su - postgres

# 启动postgresql命令行程序
psql

# 初始化数据库
initdb;

# 退出postgresql命令行程序
\q

# 退出postgres用户
exit
```

## 启动数据库

* 设置开机启动
  systemctl enable postgresql

* 启动数据库
  systemctl start postgresql
